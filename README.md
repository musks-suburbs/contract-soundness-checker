# contract-soundness-checker
Простой CLI-инструмент для проверки "звуковости" (soundness) смарт-контрактов: вычисляет keccak-хэш байткода on-chain и сравнивает его с эталоном. Полезно для контроля неизменности L1-контрактов, связанных с приватными L2/zk проектами (Aztec, Zama), а также для общих задач Web3-аудита
